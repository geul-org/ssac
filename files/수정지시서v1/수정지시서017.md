# 수정지시서017: DDL 파서 @archived 태그 지원

## 배경

DDL → OpenAPI/SSaC 역방향 교차 검증을 fullend에서 구현하려 한다. "DDL 테이블/컬럼이 있는데 API에서 안 쓰이면 WARNING"을 내되, 과거에 쓰다가 지금은 안 쓰지만 고객 데이터 보존을 위해 남겨둔 테이블/컬럼은 제외해야 한다.

이를 위해 DDL SQL 파일에 `@archived` 태그를 지원하고, SymbolTable에 해당 정보를 노출한다.

## 사용법

```sql
-- 테이블 수준: 테이블 전체가 보존 대상
-- @archived
CREATE TABLE legacy_notifications (
    id BIGSERIAL PRIMARY KEY,
    message TEXT
);

-- 컬럼 수준: 특정 컬럼만 보존 대상
CREATE TABLE courses (
    id BIGSERIAL PRIMARY KEY,
    title VARCHAR(255) NOT NULL,
    old_category VARCHAR(100), -- @archived
    category VARCHAR(50) NOT NULL
);
```

## 수정

### 1. `validator/symbol.go` — SymbolTable, DDLTable 구조체

```go
type SymbolTable struct {
	Models         map[string]ModelSymbol
	Operations     map[string]OperationSymbol
	Funcs          map[string]bool
	DDLTables      map[string]DDLTable
	DTOs           map[string]bool
	ArchivedTables map[string]bool // @archived 테이블 (e.g. "legacy_notifications" → true)
}

type DDLTable struct {
	Columns         map[string]string
	ColumnOrder     []string
	ForeignKeys     []ForeignKey
	Indexes         []Index
	ArchivedColumns map[string]bool // @archived 컬럼 (e.g. "old_category" → true)
}
```

### 2. `validator/symbol.go` — `LoadSymbolTable` 초기화

`ArchivedTables` 초기화 추가:

```go
st := &SymbolTable{
	// ... 기존 필드 ...
	ArchivedTables: make(map[string]bool),
}
```

### 3. `validator/symbol.go` — `parseDDLTables` 함수

테이블 수준 `@archived`:
- `CREATE TABLE` 직전 줄에 `-- @archived` 주석이 있으면 `ArchivedTables[tableName] = true`

컬럼 수준 `@archived`:
- 컬럼 정의 줄 끝에 `-- @archived`가 있으면 `DDLTable.ArchivedColumns[colName] = true`

```go
func parseDDLTables(content string, tables map[string]DDLTable, archivedTables map[string]bool) {
	lines := strings.Split(content, "\n")
	var currentTable string
	prevLineArchived := false

	for _, line := range lines {
		trimmed := strings.TrimSpace(line)
		upper := strings.ToUpper(trimmed)

		// -- @archived 주석 감지 (다음 줄의 CREATE TABLE에 적용)
		if strings.Contains(trimmed, "@archived") && strings.HasPrefix(trimmed, "--") {
			prevLineArchived = true
			continue
		}

		if strings.HasPrefix(upper, "CREATE TABLE") {
			// ... 기존 테이블명 파싱 ...
			if prevLineArchived {
				archivedTables[currentTable] = true
			}
			prevLineArchived = false
			continue
		}

		prevLineArchived = false // CREATE TABLE이 아닌 다른 줄이 오면 리셋

		// 컬럼 라인에서 인라인 -- @archived 감지
		if currentTable != "" && strings.Contains(line, "-- @archived") {
			// 컬럼명 추출 후
			if t, ok := tables[currentTable]; ok {
				if t.ArchivedColumns == nil {
					t.ArchivedColumns = make(map[string]bool)
				}
				t.ArchivedColumns[colName] = true
				tables[currentTable] = t
			}
		}

		// ... 기존 컬럼 파싱 로직 ...
	}
}
```

**주의**: `parseDDLTables` 시그니처에 `archivedTables map[string]bool` 파라미터 추가. 호출부(`loadDDL`)도 수정.
