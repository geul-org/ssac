# 수정지시서 001: generator/go_target.go 파일 분리 리팩토링

## 배경

`go_target.go` 1,536줄 단일 파일에 코드젠 로직 전체가 몰려있음. 맥락 유지 불가로 BUG005~010 같은 케이스 누락이 반복 발생.

## 목표

동작 변경 없이 파일만 분리. `go test ./generator/...` 전체 통과 필수.
