# 수정지시서 010 실행결과

## 수정 내용

`generator/go_target.go`의 `buildJSONBodyParams`에서 json tag 생성 시 `toSnakeCase()` 호출을 제거하고 원본 필드명(`rp.name`)을 그대로 사용하도록 변경.

```go
// 수정 전
jsonTag := toSnakeCase(rp.name)
buf.WriteString(fmt.Sprintf("\t\t%s %s `json:\"%s\"`\n", rp.name, rp.goType, jsonTag))

// 수정 후
buf.WriteString(fmt.Sprintf("\t\t%s %s `json:\"%s\"`\n", rp.name, rp.goType, rp.name))
```

## 변경 파일

| 파일 | 변경 |
|---|---|
| `generator/go_target.go` | `toSnakeCase(rp.name)` → `rp.name` (1줄 수정) |
| `generator/generator_test.go` | `TestGenerateTypedRequestParams` json tag 기대값 변경: `start_at` → `StartAt`, `end_at` → `EndAt` |

## 테스트

60개 전체 통과 (parser 16 + validator 33 + generator 11).

## 생성 코드 변화

```go
// 수정 전
var req struct {
    StartAt time.Time `json:"start_at"`
    EndAt   time.Time `json:"end_at"`
}

// 수정 후
var req struct {
    StartAt time.Time `json:"StartAt"`
    EndAt   time.Time `json:"EndAt"`
}
```

OpenAPI 스펙의 필드명과 json tag가 일치하여 클라이언트 요청이 정상 디코딩됨.
